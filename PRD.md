# 文件传输系统产品需求文档

## 产品概述
基于Web的跨平台文件传输解决方案，提供安全便捷的文件传输和剪贴板同步功能，支持桌面端和移动端访问。

---

## 核心功能列表
### 1. 多文件传输
- **格式支持**：视频(mp4/mov/avi等)、图片(jpg/png/gif等)
- **容量限制**：
  - 单文件最大：1GB
  - 单次最多：6个文件
- **传输特性**：
  - 实时进度反馈
  - 自动重试机制
  - 下载链接直给

### 2. 剪贴板同步
- **内容类型**：
  - 纯文本（自动保存）
  - 截图/图片（自动解析）
- **历史记录**：
  - 自动追加存储
  - 时间戳记录
  - 永久保存至`paste.txt`

### 3. 设备兼容
- **响应式布局**：
  - 移动端触控优化
  - 文件选择器适配
  - 自动缩放界面元素
- **跨平台支持**：
  - macOS
  - Windows
  - iOS/Android
